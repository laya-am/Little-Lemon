**Installation & Run Locally**

Clone the project:
  
  git clone https://github.com/laya-am/Little-Lemon
  
  
Go to the project directory:
  
  cd Little-Lemon
  
  
Install dependencies:
  
  npm install
  
  
Start the server:
  
  npm start
  
  
Open http://localhost:3000 to view it in the browser.
